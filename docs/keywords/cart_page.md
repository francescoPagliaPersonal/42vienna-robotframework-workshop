# Cart Page Keywords

_Keywords for interacting with the SauceDemo shopping cart page._

**Source:** `resources/cart_page.resource`

## Keywords (5)

### Cart Should Be Empty

Verify that the cart contains no items.

---

### Cart Should Contain

Verify that a product with the given name is in the cart.

**Arguments:**

| Name | Type | Default |
|------|------|---------|
| `product_name` |  | _required_ |

---

### Continue Shopping

Click the Continue Shopping button to return to the products page.

---

### Open Cart

Navigate to the shopping cart page.

---

### Proceed To Checkout

Click the Checkout button on the cart page.

---
